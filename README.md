# FF4FE-Entrance-Rando-Tracker
Bizhawk Tool for Entrance rando Auto-tracking
Add FF4FE_ERTracker.dll to your ExternalTools folder in Bizhawk
If you choose to build it yourself
Navigate to folder containing FF4FE_ERTracker.cs and FF4FE_ERTracker.csproj via Powershell
run dotnet build -c Release
copy file FF4FE_ERTracker.dll from ~directory/bin/release/net48 to 'PathToBizhawk'/ExternalTools
